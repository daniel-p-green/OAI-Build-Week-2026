# WorkshopLM

WorkshopLM is a local Capture → Map → Brief → Create workflow. The four stages are a compact index of the work, not a checklist of every feature. Use native ChatGPT/Codex conversation for thinking and voice; open the local browser Workshop for the semantic Map, Brief review, Style, created work, and Storyboard.

- **Capture:** Conversation and Sources live here. Add sanctioned local files or URLs, then use `search` and `fetch` to inspect normalized source evidence.
- **Map:** examine the auto-organized semantic canvas for hierarchy, evidence clusters, synthesis, direction, and a recommended path. Edit the meaning and relationships without turning the Map into a generic whiteboard.
- **Brief:** use `Approve brief` only when the current Map is grounded and ready. `FRAME.md` stores the executable Brief.
- **Create:** choose or derive Style, then create the selected professional work. The Storyboard stays editable, and `Create video` remains unavailable until the current Storyboard is approved.

Every grounded claim must retain a source locator. The compact plugin surface is only a doorway; use the browser Workshop to edit the Map, review Sources, and work with Outputs.
