# WorkshopLM

WorkshopLM is a local Capture → Shape → Deliver workflow. Use native ChatGPT/Codex conversation for thinking and voice; open the local browser workspace for the Sources rail, editable Map, style review, Studio, storyboard, and provenance.

- **Capture:** add sanctioned local files or URLs, then use `search` and `fetch` to inspect normalized source evidence.
- **Shape:** examine the cited Map and approve it as the brief only when the current Map is grounded and ready. `FRAME.md` is the executable brief.
- **Deliver:** choose one output type. A storyboard remains editable; video rendering is blocked until its current version is approved.

Every grounded claim must retain a source locator. The compact plugin surfaces only show status and trace; use the browser workspace for Map editing and production work.
