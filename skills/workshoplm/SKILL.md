# WorkshopLM

WorkshopLM is a local Capture → Shape → Deliver workflow. Use native ChatGPT/Codex conversation for thinking and voice; open the local browser Workshop for Sources, the editable Map, Brief and Style review, visual Outputs, and the Storyboard.

- **Capture:** add sanctioned local files or URLs, then use `search` and `fetch` to inspect normalized source evidence.
- **Shape:** examine the cited Map and use `Approve brief` only when the current Map is grounded and ready. `FRAME.md` stores the executable Brief.
- **Deliver:** use `Create outputs` to make the selected professional work. The Storyboard stays editable, and `Create video` remains unavailable until the current Storyboard is approved.

Every grounded claim must retain a source locator. The compact plugin surface is only a doorway; use the browser Workshop to edit the Map, review Sources, and work with Outputs.
