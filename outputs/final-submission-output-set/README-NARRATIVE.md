# WorkshopLM submission narrative

## Promise

WorkshopLM is the professional knowledge workspace that turns conversations and source material into presentations, graphics, Audio Overviews, visual Maps, Storyboards, and Videos. Every expression shares the same knowledge, visual identity, and connection to its Sources.

## Product path

1. **Capture:** speak naturally or add notes, websites, and local documents. Conversation and Sources stay together.
2. **Map:** review the auto-organized hierarchy, evidence clusters, synthesis, and recommended path; edit the grounded semantic canvas directly.
3. **Brief:** approve the current Map as the creation direction.
4. **Create:** choose a website-derived, saved, or manually specified Style, then create a Presentation, Infographic, Audio Overview, coherent Image set, and editable Storyboard. Video appears contextually after Storyboard approval.

WorkshopLM owns the professional's Conversation, Sources, Map, Brief, Style, created work, Storyboard, and exact Source trace. Every format is a first-class expression of the same Workshop. Codex is the development and launch host, not the professional's chat surface. ChatGPT Work parity is not claimed.

## What this package contains

The package was built from the same approved Workshop state as the demo. Its manifest binds the Source scope, claim IDs, Map revision, Brief and Style versions, Storyboard approval, file hashes, and dependency fingerprint for the Devpost copy, this repository narrative, Presentation, Infographic, Audio Overview, Image manifest, styled public cover, two Video proof thumbnails, Storyboard, narration, evidence report, build trace, and Video.
